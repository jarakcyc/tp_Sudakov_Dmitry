Основне исходный код находится в 3-х файла:
1) ciphers.py - описание шифров(Цезаря, Виженера, Вернама)
фунции encode и decode для каждого
2) hack.py - все, что связано с расшифровкой без знания ключа.
(Цезаря, Виженера)
В свзяи с добавлением в hack шифра Виженера, немного изменил
парс аргументов для субпарсера 'hack'. Добавил
аргумент --cipher.
3) encryptor.py - парс аргументов и запуск соответсвтующих
функций

В файле tester.py находятся некоторые тесты для написанных 
функций. 

master_and_margarita.txt - текст для тренировок
text_1.txt, text_2.txt - я использовал для тестирования hack
